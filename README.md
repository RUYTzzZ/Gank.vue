### 基友 vue + vue-router + webpack4 + elementUI
- 配置文件在build目录下
- 打包后的文件在build的webpack目录下

### 安装
1. fork 或者clone代码
```
git clone https://github.com/GuMengYu/webpack4Demo.git
```
2. 切换到 dev分支
```
git checkout dev
```
3. 安装依赖
```
npm i --no-save
```
4. 开发环境运行
```
npm run dev
```
5. 打包
```
npm run prod
```
