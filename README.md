### 基友 vue + vue-router + webpack4 + elementUI
- 配置文件在build目录下
- 打包后的文件在build的webpack目录下

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Lints and fixes files
```
yarn run lint
```
